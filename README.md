ofxLytroFileTools
=================

Tools to parse and view the Lytro camera (http://www.lytro.com/)  files in openFrameworks

REQUIRES:
openFrameworks 0071 http://www.openframeworks.cc/

TESTED ON:
Mac OSX 10.6.8
Xcode 3.2.6

MISC:
Based off of lfptools (Thanks!) https://github.com/Sittro/lfptools
